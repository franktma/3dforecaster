About 3-D Printing Forecaster


Overview

3-D Printing Forecaster is a statistical model that predicts the 3-D printing time for a given model part based on its geometric features.

You can find a powerpoint presentation of the model here. (Note that since the slides contain private data from Paperless Parts, it is not completely open to the public. But you can contact me and I would happy to give you access if you agree to keep the content confidential.)

Data

The data is taken from Thingi10k, a public online database of ten thousand part models with high complexity but suitable for 3-D printing.

Training

The labeled target is the printing time given by 3-D printing simulation using the open source simulation package Cura Software by Ultimaker. The features are extracted using the open source Python library, Trimesh for loading and using 3-D part model (.stl) files. The features include standard geometric features from Trimesh, including: Volume, Surface Area, Bounding Box Volume, height. In addition a privately engineered variable was extracted that helps to gage how complex the model part is for 3-D printing. The model was trained on 4/5 of the dataset. It composes of two stages: First, the parts are classified into two groups using a logistic classifier that estimates how complex is the part for pring. Next, a weighted linear regression is performed on each of the two groups.

Valdiation

The model was tested on independent subsets of the dataset using a 10-fold cross validation.
